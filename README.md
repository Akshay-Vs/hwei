# HWEI

### Tech stack

- **Backend**: NestJS, PostgreSQL, Prisma, Redis, RabbitMQ/Kafka, AWS S3.
- **Frontend** (Admin Panel): Next.js, Tailwind CSS/MUI, Recharts/Chart.js.
- **DevOps**: Docker, Kubernetes, AWS, Vercel.
- **Monitoring**: Sentry, Datadog, Winston.
